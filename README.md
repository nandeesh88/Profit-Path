# Profit Path – FinTech SaaS Platform

Profit Path is a modern FinTech SaaS web application designed with a Minimalist + Soft Industrial aesthetic. It enables users to track expenses, analyze financial performance, explore investment options, and plan financial goals within a clean, glassmorphism-based interface.

---

## Demo Access

Use the demo account to explore the application:

Email: demo@example.com  
Password: any password works  

Authentication is session-based and protects all dashboard routes.

---

## Core Features

### 1. Landing Page (/)
- Hero section with product overview  
- Feature cards for Expenses, Dashboard, Investments, and Goals  
- Redirects to login before accessing protected pages  

### 2. Dashboard (/dashboard)
- Financial KPIs (Balance, Income, Expenses, Savings)  
- Spending trend chart  
- Expense category pie chart  
- Budget progress tracking  

### 3. Expense Tracking (/expenses)
- Add and delete transactions  
- Category-based tracking (Food, Rent, Utilities, etc.)  
- Summary statistics (total, average, monthly expenses)  
- Preloaded dummy data  

### 4. Investment Hub (/investments)
- Multiple investment options  
- Risk-level filtering (Low, Medium, High)  
- AI-based recommendations  
- Investment details with expected returns  

### 5. Goal Planning (/goals)
- Predefined financial goals  
- Progress tracking with percentages  
- Priority categorization  
- Add new goals functionality

## Design System

- Background: Off-White (#F9FAFB)  
- Primary Text: Charcoal (#1A1A1A)  
- Accent: Sage Green  
- Warning: Terracotta  
- Typography: Plus Jakarta Sans  
- UI Style: Glassmorphism, soft shadows, responsive layout  
- Dark Mode supported

## Installation & Setup

Clone the repository:

```bash
git clone https://github.com/nandeesh88/Profit-Path.git
cd Profit-Path
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

http://localhost:3000
